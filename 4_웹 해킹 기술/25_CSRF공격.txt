25_CSRF공격

CSRF(Cross Site Request Forgery)
사이트 간 요청 위조
피싱을 활용해 사용자 모르게 패스워드 변경!
옥션 해킹 사건에서 사용된 공격기법

					사  용  자  ---------------
											  | 2. 피싱
					|\       | 3. 패스워드    |
1. 정상접속, 로그인	|        |    변경요청   해커
				   \|        |/               |
					                          | 4. 변경된 패스워드로 접속
					웹  서  버	---------------
					
1. 사용자가 사이트에 정상 로그인
2. 로그인 되어있는 상태에서 해커가 피싱 메일을 통해 링크를 클릭하도록 함
3. 클릭한 시점에 로그인 되어있던 사이트에 패스워드를 해커가 지정한 패스워드로 변경하는 요청이 자동으로 전송
4. 변경된 패스워드를 이용하여 해당 사용자 아이디 사용

=> 사용자가 피싱에 당하여 링크를 클릭해야 하고 사용자가 로그인 되어있는 상태여야한다
